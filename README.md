# 🍽️ Smart Restaurant Recommendation & Cuisine Classification System

## 📌 Overview
This project presents a Machine Learning-based solution that combines:
- Restaurant Recommendation System (Content-Based Filtering)
- Cuisine Classification Model (Supervised Learning)

The system leverages restaurant data to deliver personalized recommendations and predict cuisine categories based on key features such as pricing, ratings, and user engagement.

---

## 🎯 Objectives
- Build a recommendation system based on user preferences  
- Develop a classification model to predict restaurant cuisines  
- Apply end-to-end machine learning workflow  
- Evaluate model performance using standard metrics  

---

## 🚀 Features
- Personalized restaurant recommendations  
- Cuisine classification using machine learning  
- Data preprocessing and cleaning  
- Feature encoding and transformation  
- Model training and evaluation  
- Performance metrics (Accuracy, Precision, Recall, F1-score)  

---

## 🧠 Machine Learning Techniques
- Content-Based Filtering
- One-Hot Encoding
- Label Encoding
- Random Forest Classifier
- Cosine Similarity
- Train-Test Split

---

## 📂 Dataset
The dataset contains restaurant-related information:
- Restaurant Name  
- Cuisines  
- Average Cost for Two  
- Aggregate Rating  
- Votes
